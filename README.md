### How to install it
First install pip

Second install dependencies for this project
```
bottle==0.12.13
certifi==2018.4.16
chardet==3.0.4
idna==2.7
Pillow==5.2.0
requests==2.19.1
urllib3==1.23
```

run Rest-API
python3 main.py

and then using index.html join it =)

######this code tested on Python 3
